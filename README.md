# Disease Outbreak Severity Predictor

An AI-powered early warning system that predicts whether a disease outbreak in India will become **SEVERE** or **NON-SEVERE**.

## 🌐 Live Demo
Try the app here: [diseaseoutbreak-predictor.streamlit.app](https://diseaseoutbreak-predictor.streamlit.app/)

## 📊 About the Project
- **Model:** Random Forest Classifier
- **Accuracy:** 78.1%
- **Data Source:** EpiClim dataset (2009-2022)
- **Coverage:** 791 districts across India
- **SDG Alignment:** SDG 3 - Target 3.d (Early warning systems)

## 🚀 Features
- Predicts severity for 5 major diseases (Malaria, Dengue, Cholera, Chikungunya, Acute Diarrhoeal Disease)
- Uses climatic factors (temperature, precipitation, season)
- Provides actionable recommendations for health authorities

## 🛠️ Tech Stack
- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy, Matplotlib

## 📂 Repository Structure
- `app.py` - Main Streamlit application
- `requirements.txt` - Python dependencies
- `severity_model_final.pkl` - Trained Random Forest model
